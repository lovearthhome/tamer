### Tamer(驯兽师)
Tamer是爱蓝地球工作室易明雨开发的人体步态识别辅助学习APP.本APP用Ionic+angularJs+cordova架构开发。开发语言主要为javascript。本APP主要用来帮助人体步态识别的初学者对步态行为有初步感性的了解。

##功能包括：
1：采集步态数据，绘制出波形。
2：对数据进行FFT操作，绘制出频谱。
3：把样本数据上传到步态云服务器里。
4：浏览云服务器里各种人的步态数据（不同年龄段，体重，时间段）。
